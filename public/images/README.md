# Folder Images

## Instruksi Upload Foto Header

Untuk mengubah background header halaman Informasi:

1. **Simpan foto upacara bendera** (atau foto lain yang Anda inginkan) ke folder ini
2. **Rename foto** menjadi: `header-info.jpg`
3. **Refresh halaman** informasi di browser

## Spesifikasi Foto yang Disarankan:
- Format: JPG, PNG, atau WEBP
- Resolusi minimal: 1920x600 px
- Ukuran file: Maksimal 2MB untuk performa optimal
- Orientasi: Landscape (horizontal)

## Catatan:
- Foto akan ditampilkan dengan overlay gradient biru-hijau untuk keterbacaan teks
- Pastikan foto memiliki kontras yang baik
- Foto dengan subjek di tengah atau kanan akan terlihat lebih baik
